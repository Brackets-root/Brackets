About This Site

I created this website as a personal playground to experiment with web design, animations, and interactivity. It's a space where I can build, test, and break things safely while learning front-end development.

Along the way, I'm also exploring how to make a site user-friendly and visually engaging, all while practicing my skills in JavaScript, CSS, and HTML.

This project is both a portfolio of my experiments and a step in my journey towards understanding web technologies more deeply.
